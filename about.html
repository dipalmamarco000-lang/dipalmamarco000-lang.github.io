Chi Sono

La Mia Storia. E perché amo ciò che faccio.

Mi chiamo Marco Dipalma, un Junior Ux Designer. La mia passione è trasformare idee complesse in interfacce utente intuitive e funzionali. Ho iniziato il mio percorso nel mondo del web quando grazie a Start2Impact ho scoperto le potenzialità dei linguaggi di programmazione e dei tool grafici.

Durante il mio percorso di studi, ho avuto l'opportunità di lavorare su una varietà di progetti, dai siti e-commerce responsivi alle applicazioni web interattive. Credo fermamente nell'importanza di un codice pulito, nell'attenzione ai dettagli e nell'apprendimento continuo.

Le Mie Competenze. Cosa posso fare per te.

Le mie principali competenze includono:

  • HTML5 & CSS3: Struttura semantica e stili moderni, inclusi Flexbox e Grid.
  • Sass/Less: Pre-processori CSS per una gestione più efficiente degli stili.
  • JavaScript: Interattività e dinamismo per il web.
  • Framework Front-End: Bootstrap 5, Materialize, React.
  • Responsive Design: Assicurare che i siti siano perfetti su ogni dispositivo.
  • Version Control: Git & GitHub per una collaborazione efficiente.
  • Suite Adobe: Photoshop, Illustrator, InDesign, ecc.
  • Tool grafici: Figma, Figjam, Canva, Sketch, Miro.
